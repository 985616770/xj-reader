# 玄机书城

- 实现了阅读，搜索，听书，书籍详情，书架功能

- `LocalStorage` 和 `indexDB` 进行存储数据

- 兼容复杂手势和鼠标的操作，书签手势操作。

- **`Vuex`** + **`mixin`** + **`slot`** 的使用，解决组件解耦，复用更高效，精简了代码量。

- **`vue-i18n`** 实现国际化。

- 完成开发上线全流程。

- 纯css3实现随机推荐书籍动画。
